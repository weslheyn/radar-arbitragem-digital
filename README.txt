RADAR ARBITRAGEM DIGITAL - MVP

Como testar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Cadastre um produto, coloque preço de compra/venda e veja lucro, margem e score.

Observação:
- Os dados ficam salvos no navegador via localStorage.
- Para instalar como PWA no celular, hospede a pasta em um servidor HTTPS ou GitHub Pages.
- O Facebook Marketplace está no modo publicação assistida: o app gera o anúncio e abre a página para você finalizar manualmente.

Próximas evoluções:
- Firebase para sincronizar produtos entre dispositivos.
- Login de usuário.
- Integração Mercado Livre via API.
- Integração com planilha Google Sheets.
- Alertas WhatsApp/Telegram.
