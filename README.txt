RADAR ARBITRAGEM DIGITAL PRO V2

Novidades:
- Radar automático profissional
- Consulta preço de mercado no Mercado Livre via API pública
- Base inicial de fornecedores/nichos para cruzamento automático
- Filtros de margem mínima e lucro mínimo
- Geração automática de oportunidades
- Oportunidades exemplo para testar sem internet/API
- Gerador de anúncio, WhatsApp, OLX e Facebook Marketplace assistido

Como usar:
1. Abra index.html ou publique no GitHub Pages.
2. Na área "Radar automático profissional", digite um nicho: fone bluetooth, smartwatch, mini camera wifi, carregador etc.
3. Clique em "Rodar radar automático".
4. O app cria oportunidades automaticamente se lucro/margem passarem nos filtros.

Observação:
Esta versão evita automações arriscadas. Facebook Marketplace continua em modo assistido.
Para busca 100% profissional com fornecedores reais, a próxima etapa é conectar APIs oficiais e/ou planilha própria de fornecedores.
